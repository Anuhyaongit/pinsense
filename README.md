# pinsense

📌 PinSense — Smart Pinterest Board Organizer 📌

A Pinterest inspired smart board organizer that uses computer vision and aesthetic analysis to automatically group and tag a user’s Pins.  It helps users discover patterns in their taste, clean up messy boards, and receive personalized recommendations based on their visual style. PinSense combines machine learning, design thinking, and full‑stack engineering to solve a real problem Pinterest users face: keeping their creativity organized.

**Core Features**

✯ Visual Embedding Extraction — Convert images into vector representations using CLIP

✯ Smart Clustering — Group similar pins into themes (e.g., “minimalist decor”, “streetwear”, “boho interiors”)

✯ Aesthetic Tagging — Identify color palettes, styles, and moods

✯ Board Suggestions — Recommend new board names and categories

✯ Personalized Recommendations — Suggest new pins based on the user’s visual taste

✯ Clean, Pinterest‑inspired UI — A visual experience that feels familiar and intuitive
